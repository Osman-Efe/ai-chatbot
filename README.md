🤖 Gemini AI Chatbot Projesi
Bu proje, Google'ın Gemini 1.5 Flash modelini kullanan, modern, kullanıcı dostu ve duyarlı (responsive) bir web tabanlı chatbot uygulamasıdır. Kullanıcıların hem metin hem de görsel (image) üzerinden yapay zeka ile etkileşime girmesine olanak tanır.

✨ Özellikler
Gemini API Entegrasyonu: gemini-flash-latest modeli ile hızlı ve zeki yanıtlar.

Görsel Analizi: Fotoğraf yükleme desteği sayesinde görseller hakkında soru sorabilme.

Emoji Desteği: Emoji Mart kütüphanesi ile zengin emoji seçici.

Dinamik UI:

Yazma efektli "Düşünüyor..." göstergesi (Thinking indicator).

Mesaj uzunluğuna göre otomatik genişleyen giriş alanı.

Tamamen responsive (mobil uyumlu) tasarım.

Modern Tasarım: Inter font ailesi ve Material Symbols ikonları ile şık bir kullanıcı deneyimi.

🛠️ Kullanılan Teknolojiler
Frontend: HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+).

API: Google Generative AI (Gemini).

Kütüphaneler: Emoji Mart (Emoji seçici için).

🚀 Kurulum ve Çalıştırma
